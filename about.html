Welcome to Snippet

Snippet is a free modern CSS boilerplate for your next project. It supports printing, dark-mode, and CSS variables. It'll work with people on Safari and Chrome. It's incredibly lightweight, and based on the very well made but lacking modern features boilerplate: Skeleton. Just like Skeleton, Snippet is open-source.

Install

Try with Composer

													composer require snippet

Or NPM

													npm install snippet
